# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-02-07
### Added
- Initial monorepo project structure.
- NestJS Core API with Prisma schema.
- FastAPI Matching Service with keyword matching algorithm.
- Next.js 14 Frontend with Dashboard UI.
- Docker Compose infrastructure (Postgres, Redis, MinIO).
- Shared package for types and schemas.
- Makefile and CI workflow templates.
